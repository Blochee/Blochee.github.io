Numpy

作用：提供了一种数组类型，能够进行科学计算

导入方法： import numpy as np

显示版本：np._version_

注意：Numpy默认ndarray的所有的元素的类型是相同的

​ 如果传进来的列表中包含不同的数据类型，则统一为同一类型

​ 优先级：str>float>int

创建Array

  1. 使用np.array()
age = [15,16,18]
array3 = np.array(age, dtype=np.float32)
  1. 使用np.routines()创建
  • np.ones(shape, dtype=None, order=’C’)
# 构造一个5行3列的二维数组
np.ones(shape=(5,3), dtype=np.int8)
  • np.zeros(shape, dtype=float, order=’C’)
np.zeros(shape=(5,6))
  • np.full(shape, fill_value, dtype=None, order=’C’)
np.full(shape=(2,3), fill_value=6)
  • np.eye(N, M=None, k=0, dtype=float)
# 生成对角线元素为1的单位矩阵
np.eye(N=3)
  • np.linspace(start, stop, num=50, endpoint=True, retstep=False, dtype=None)
# 等差数列
np.linspace(0,10,10)
  • np.arange([start, ]stop, [step, ]dtype=None)
np.arange(0,10,step=1)
  • np.random.randint(low, high=None, size=None, dtype=’l’)
# size 同 shape
np.random.randint(0,100,size=(5,5))
  • 正态分布函数：np.random.randn(d0, d1, …, dn) 标准正态分布，np.random.normal() 普通正态分布
np.random.randn(3,5)
np.random.normal(loc=170, scale=5, size=(5,3))
  • np.random.random(size=None):生成0到1的随机数，左闭右开
np.random.random(size=(5,5))
# 生成一组随机索引
np.random.permutation(10)

ndarray的属性

四个重要参数：

  • ndim：维度
  • shape：形状
  • size：总长度
  • dtype：元素类型
arr = np.random.randint(0,100,size=(5,4,3))
arr.ndim
arr.shape
arr.size
# 获取数组元素类型
arr.dtype
# 获取数组类型
type(arr)

ndarray的基本操作

  1. 索引
arr1 = np.array([1,2,3,4,5])
arr1[0]
# 使用列表作为索引访问
arr1[[0,1,2,0,1,2]]

arr2 = np.array([[1,2,3],[4,5,6]])
# 列表的访问方式
arr2[0][0]

# 使用BOOL列表作为索引访问, True对应的值会被返回
bool_list = [True, False, True, False, True]
arr1[bool_list]
  1. 切片
arr2 = np.random.randint(0,100,size=(5,6))
# 行切片
arr2[0:2]
# 列切片 不论多少维，每一个维度的切片范围都是用冒号表示，使用逗号分割，最后一个维度可以省略，但是被切片的维度之前的维度不能省
arr2[:,0:2]

arr3 = np.random.randint(0,10,size=(3,4,5))
# 切前两个数组
arr3[0:2]
# 切割每一个二维数组的前两列
arr3[:,:,0:2]
# 切割每一个二维数组的前两行
arr3[:,0:2]
#数据反转

arr1 = np.array([1,2,3,4,5])
arr1[::-1]
  1. 变形
arr = np.random.randint(0,10,size=(20))
#参数是一个元组
arr.reshape((4,5))
arr.reshape(3,7)
  1. 级联
np.concatenate((arr1, arr2), axis=1)

  1. 切分

  2. 副本

ndarray的聚合操作

  1. 常见操作
arr = np.random.randint(0,10,size=10)
arr.sum()
# 求平均值
arr.mean()
arr.max()
arr.min()
# 求最大值索引
arr.argmax()
arr.argmin()
# 标准方差
arr.std()
# 求方差
arr.var()
# 注意这个语法格式
np.median(arr)
#这是一个错误的格式
arr.median()

ndarray矩阵操作

  1. 基本的矩阵操作
  • 算术运算符：加减乘除
  • 矩阵积np.dot
a = np.array([[1,2],[3,4]])
b = np.array([[1,1],[2,2]])
display(a, b)

np.dot(a, b)
  1. 广播机制
arr = np.array([1,2,3,4,5])
data = 3
arr + data

注意：ndarray广播机制的两条规则

  • 为缺失的维度补1
  • 假定缺失元素用已经有的值填充
m = np.ones((2,3))
a = np.arange(3)
display(m, a)

m + a
m + 3

# 不能广播运算的例子
m = np.ones(shape=(3,3))
n = np.array([[1,2],[3,4]])

a = np.arange(0,3,step=1).reshape((3, 1))
b = np.arange(3)
a + b

ndarray的排序

  1. 快速排序
data = np.random.permutation(10)

data.sort()
data

data.sort(data)
  1. 部分排序
data = np.random.permutation(10000)

np.partition(data, 4)[:4]
np.partition(data,-4)[-4:]

np.partition(a,k)

  • 当k为正数时，我们想要得到最小的k个数
  • 当k为负数时，我们想要得到最大的k个数
